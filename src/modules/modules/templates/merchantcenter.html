Conditions préalables

Google Merchant Center permet d'importer des produits sur Google et les proposer dans Google Shopping ainsi que dans d'autres services Google.

Afin de configurer correctement l'export vers Google Merchant Center, vous devez suivre la procédure suivante :

  1. Dans Google Merchant Center :
    1. Ouvrez un compte Google Merchant Center
    2. Terminez le processus d'ouverture de compte en validant l'adresse du site Internet pour en revendiquer la propriété sur la page Paramètres généraux de Google Merchant center
  2. Dans la console d'administration de Kiubi :
    1. Il est recommandé d'activer l'accès en HTTPS, sans quoi le flux pourrait être refusé par Google Merchant Center
    2. Dans le formulaire ci-dessous, choisissez les catégories de produits à exporter et activez le module
    3. Attendez le lendemain matin que le flux journalier soit généré
    4. Une fois le flux généré, un lien Adresse du flux vers Google Merchant Center est disponible. Cliquez sur ce lien pour télécharger le flux.
  3. Dans Google Merchant Center :
    1. Allez dans Flux de données et cliquez sur le lien Nouveau flux de données test
    2. Indiquez le pays et un nom de flux (flux_de_test.txt, par exemple). Enregistrez les modifications
    3. Cliquez sur le lien Importation manuelle en face du flux de test et sélectionnez le flux précédement téléchargé. Cliquez sur Importer et traiter ce fichier
    4. Attendez que Google traite le flux (l'opération peut prendre plus d'une heure)
    5. Une fois l'opération terminée, vérifiez que le flux a été importé sans erreur. Dans le cas contraire, cliquez sur Afficher les erreurs et suivez les corrections proposées
    6. UNIQUEMENT si l'importation s'est faite sans aucune erreur, créez un nouveau flux, mais pas de test cette fois
    7. Cliquez sur le lien Créer un planning. Importez le flux tous les jours à 6h. Indiquez dans le champ URL du flux l'adresse du flux vers Google Merchant Center
    8. Cliquez sur le bouton Planifier et récupérer

Aucune autre intervention n'est nécessaire, si toutes les opérations ont été correctement effectuées, Google Merchant Center récupéra le flux de produits tous les jours.

Catégories à inclure dans le flux

Sélectionner les catégories de produits du catalogue à inclure dans le flux vers Google Merchant Center. Pour chaque catégorie, vous pouvez indiquer à quelle catégorie de Google Merchant Center elle correspond. Utiliser le moteur de recherche pour trouver la catégorie la plus pertinente.

La correspondance est strictement obligatoire pour les produits de type vêtements, livres, médias (DVD, musique, ...) et jeux vidéo, sous peine de voir Google pénaliser ou refuser les produits. Elle reste fortement recommandée pour tous les autres types de produits. Pour plus d'informations, consulter le centre d'aide Google Merchant Center.
