Conditions préalables

ATOS SIPS est la solution de paiement en ligne sécurisé proposée par les banques suivantes : Société Générale - Sogénactif / Agora, Crédit Lyonnais - Sherlock, HSBC, CCF - Elysnet, Crédit du Nord - Webaffaires, La Poste - Scellius, Crédit Agricole - e-Transactions, BNP Paribas - Merc@net. Pour plus d'informations, rendez-vous sur le site officiel ou prenez contact avec un établissement bancaire.

Afin de configurer correctement ce mode de paiement vous devez :

  1. Prendre contact avec un établissement bancaire afin d'obtenir l’ouverture d’un contrat monétique de Vente A Distance (VAD)
  2. Une fois que vous avez eu confirmation du service technique de l'établissement bancaire que le TPE a été activé, complétez le formulaire ci-dessous avec les informations fournis par l'établissement bancaire et activez le paiement

Aucune autre intervention technique n’est nécessaire, si toutes les opérations ont été correctement effectuées, ce mode de paiement sera alors actif sur le site. Une fois toutes ces opérations de configuration terminées, il est indispensable d'effectuer au moins un paiement réel sur le site pour tester son bon fonctionnement.

Paramètres du mode de paiement

Fichier nommé certif.fr.xxxxxxxxxxxxx. Remplaçez les x par un numéro de marchand. Le fichier et le numéro de marchand sont fournis par l'établissement bancaire.

Le message de confirmation s'affiche à la fin de la procédure de commande, une fois le paiement validé.
