Conditions préalables

Paypal est une solution de paiement en ligne simple et rapide à mettre en place pour recevoir des paiements. Pour plus d'informations, rendez-vous sur le site officiel.

Afin de configurer correctement ce mode de paiement vous devez :

  1. Allez sur le site de Paypal et vous ouvrir un compte. Vous aurez le choix entre plusieurs types de compte et vous devrez obligatoirement prendre un compte Professionnel (Premier ou Business). Dans le cas contraire, le système de paiement Paypal ne fonctionnera pas avec Kiubi. Si vous possédez déjà un compte Personnel, vous devrez surclasser votre compte.
  2. Souscrire à l'offre PayPal Intégral
  3. Configurez le paiement dans l'administration du compte Paypal (généralement dans l'onglet "Mon Compte" puis "Préférences") :

    Taxe / TVA :
    Vérifiez qu'aucun taux de TVA ne soit défini. La TVA est gérée directement par Kiubi

    Calculs des frais de livraison :
    Vérifiez qu'aucun coût ne soit défini. Les frais de livraison sont gérés directement par Kiubi

    Préférences de notification instantanée de paiement :
    Cochez la case et remplir le champ "URL de notification" avec l'URL suivante :
    <%- domain %>/TPEV/paypal/85135625eb18.html

    Préférences de réception de paiements sur le site :
    Activez le "Renvoi automatique" en choisissant "Oui"

    Remplir le champ "URL de renvoi" avec l'URL suivante :
    <%- domain %>/ecommerce/validation.html

    Activez le "Transfert des données de paiement" en choisissant "Oui"

  4. Complétez le formulaire ci-dessous avec l'adresse email de votre compte Paypal et activez le paiement.

Aucune autre intervention technique n’est nécessaire, si toutes les opérations ont été correctement effectuées, ce mode de paiement sera alors actif sur le site. Une fois toutes ces opérations de configuration terminées, il est indispensable d'effectuer au moins un paiement réel sur le site pour tester son bon fonctionnement.

Paramètres du mode de paiement

Le message de confirmation s'affiche à la fin de la procédure de commande, une fois le paiement validé.
