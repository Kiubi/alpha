Conditions préalables

Paypal est une solution de paiement en ligne simple et rapide à mettre en place pour recevoir des paiements. Pour plus d'informations, rendez-vous sur le site officiel.

Afin de configurer correctement ce mode de paiement vous devez :

  1. Allez sur le site de Paypal pour vous ouvrir un compte. Vous aurez le choix entre plusieurs types de compte. Vous devrez obligatoirement prendre un compte Professionnel (Premier ou Business). Dans le cas contraire, le système de paiement Paypal ne fonctionnera pas avec Kiubi. Si vous possédez déjà un compte Personnel, vous devrez surclasser votre compte
  2. Souscrivez à l'offre Paypal Intégral. L'offre est généralement associée dès l'ouverture de votre compte Professionnel
  3. Configurez le paiement dans l'administration du compte Paypal, dans "Mon Compte " puis "Paramètres du compte" :

    "Préférences de paiement" puis "TVA" :
    Vérifiez qu'aucun taux de TVA ne soit défini. La TVA est gérée directement par Kiubi

    "Paiements sur le site" puis "Notifications instantanées de paiement" puis "Choisir les paramètres IPN" :
    Cochez la case "Activé" et remplissez le champ "URL de notification" avec l'URL suivante :
    <%- notification_url %>

    "Paiements sur le site" puis "Préférences de site marchand" puis "Renvoi automatique pour les paiements sur site marchand"
    Activez et remplissez le champ "URL de renvoi" avec l'URL suivante :
    <%- domain %>/ecommerce/validation.html

    "Paiements sur le site" puis "Préférences de site marchand" puis "Transfert des données de paiement"
    Activez le transfert des données de paiement

    "Expédition" puis "Calcul des frais de livraison"
    Vérifiez qu'aucun coût ne soit défini. Les frais de livraison sont gérés directement par Kiubi

  4. Complétez le formulaire ci-dessous avec l'adresse email de votre compte Paypal et activez le paiement.

Aucune autre intervention technique n’est nécessaire, si toutes les opérations ont été correctement effectuées, ce mode de paiement sera alors actif sur le site. Une fois toutes ces opérations de configuration terminées, il est indispensable d'effectuer au moins un paiement réel sur le site pour tester son bon fonctionnement.
