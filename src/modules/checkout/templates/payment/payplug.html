Conditions préalables

PayPlug est une solution de paiement en ligne sécurisée. Pour plus d’informations, rendez-vous sur le site officiel.

Afin de configurer correctement ce mode de paiement vous devez :

  1. Créez un compte chez PayPlug.
  2. Connectez-vous dans votre console d'administration PayPlug.
  3. Allez dans "Paramètres" puis "Clés API". Séléctionnez, si ce n'est pas déjà le cas, le choix : "API".
  4. Dans l'encart Clé API, notez la clé secrète (Secret Key) de TEST.
  5. Complétez le formulaire ci-dessous en indiquant la clé secrète.
  6. Effectuez un paiement en mode TEST
  7. Faites les démarches au près de PayPlug pour activer votre compte.
  8. Une fois votre compte activé, connectez-vous dans votre centre console d'administration PayPlug et notez la clé secrète (Secret Key) de production (LIVE).
  9. Complétez le formulaire ci-dessous en indiquant la clé secrète de production.

Aucune autre intervention technique n’est nécessaire, si toutes les opérations ont été correctement effectuées, ce mode de paiement sera alors actif sur le site. Une fois toutes ces opérations de configuration terminées, il est indispensable d'effectuer au moins un paiement réel sur le site pour tester son bon fonctionnement.
