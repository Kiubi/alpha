Le thème graphique personnalisé permet de créer un look sur mesure pour votre un site Internet, en vous donnant un accès complet aux templates qui le composent. Aucune limitation technique particulière ne vous est alors imposée.

Afin de répondre aux exigences de qualité et de performance d'un site Internet professionnel, la maîtrise des standards du Web tels que le HTML, le CSS et le javascript est nécessaire. Le guide du webdesigner est à votre disposition pour vous aider à exploiter les possibilités de personnalisation offertes par Kiubi.

Activer le thème graphique personnalisé va également réinitialiser tous les fichiers de votre espace FTP, dont l'accès vous sera donné après l'activation du thème.
